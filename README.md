# Вероятность и Статистика для Data Science и Machine Learning

## Курс "От нуля до профи"

Комплексный курс по вероятности и статистике, предназначенный для специалистов по данным и машинному обучению. Курс построен на основе Jupyter Notebooks с теорией, формулами, кодом на Python и практическими упражнениями.

## Структура курса

### Модуль 1: Основы вероятности (Базовый уровень)
| № | Ноутбук | Тема |
|---|---------|------|
| 01 | `01_Probability_Fundamentals.ipynb` | Основные понятия: пространство исходов, события, аксиомы вероятности. Условная вероятность, теорема Байеса. Независимость и взаимоисключаемость |
| 02 | `02_Probability_Distributions.ipynb` | Дискретные и непрерывные случайные величины. PMF, PDF, CDF. Основные распределения: Бернулли, Биномиальное, Пуассон, Равномерное, Нормальное, Экспоненциальное |

### Модуль 2: Случайные величины и статистические меры
| № | Ноутбук | Тема |
|---|---------|------|
| 03 | `03_Random_Variables.ipynb` | Математическое ожидание, дисперсия, стандартное отклонение. Ковариация и корреляция |
| 04 | `04_Joint_Distributions.ipynb` | Совместные, маргинальные и условные распределения. Независимость случайных величин |

### Модуль 3: Описательная и выводная статистика
| № | Ноутбук | Тема |
|---|---------|------|
| 05 | `05_Descriptive_Statistics.ipynb` | Меры центральной тенденции и разброса. Визуализация данных |
| 06 | `06_Sampling_Distributions.ipynb` | Закон больших чисел (ЗБЧ). Центральная предельная теорема (ЦПТ) |
| 07 | `07_Confidence_Intervals.ipynb` | Доверительные интервалы для средних и пропорций |
| 08 | `08_Hypothesis_Testing.ipynb` | Проверка гипотез: p-значения, t-тесты, ANOVA, хи-квадрат тесты |

### Модуль 4: Продвинутая вероятность и Байесовская статистика
| № | Ноутбук | Тема |
|---|---------|------|
| 09 | `09_MLE_MAP.ipynb` | Максимальное правдоподобие (MLE) и максимальное апостериорное оценка (MAP) |
| 10 | `10_Bayesian_Statistics.ipynb` | Байесовский вывод: априорное, правдоподобие, апостериорное |
| 11 | `11_Monte_Carlo_Methods.ipynb` | Цепи Маркова и методы Монте-Карло |

### Модуль 5: Прикладная математика для ML (Профессиональный уровень)
| № | Ноутбук | Тема |
|---|---------|------|
| 12 | `12_Information_Theory.ipynb` | Энтропия, кросс-энтропия, KL-дивергенция |
| 13 | `13_Multivariate_Calculus.ipynb` | Частные производные, градиенты, якобианы |
| 14 | `14_Probability_in_Neural_Networks.ipynb` | Вероятность в нейронных сетях: PyTorch, autograd |

### Прикладные проекты
| № | Ноутбук | Тема |
|---|---------|------|
| 15 | `15_Supermarket_Analysis.ipynb` | Анализ данных супермаркета: выявление часто покупаемых вместе товаров |
| 16 | `16_AB_Testing.ipynb` | A/B-тестирование сайта: доверительные интервалы, z-тест для пропорций, размер эффекта, байесовский подход (Beta-Binomial) |
| 17 | `17_Spam_Classifier.ipynb` | Байесовский классификатор спама с нуля: теорема Байеса, MLE со сглаживанием Лапласа, precision/recall |
| 18 | `18_IT_Job_Market_USA.ipynb` | Анализ рынка труда IT в США на РЕАЛЬНЫХ данных (BLS OEWS, Census ACS API, Tax Foundation): где выше доход на руки, ниже аренда и налоги — композитный рейтинг штатов |
| 19 | `19_IT_Job_Market_Russia.ipynb` | Анализ рынка труда IT в России по регионам (hh.ru, Habr Career, ЦИАН, Росстат): НДФЛ 2025, формы занятости (наёмный/самозанятый/ИП), IT-ипотека — композитный рейтинг городов |
| 20 | `20_IT_Job_Market_EU.ipynb` | Анализ рынка труда IT в ЕС (2025/2026, Stack Overflow Survey, Numbeo, OECD Taxing Wages): подоходный налог + соц. взносы, аренда/ипотека, разные валюты — композитный рейтинг стран |
| 21 | `21_Global_IT_Comparison.ipynb` | Сводное сравнение США / Россия / ЕС в единой валюте (USD) с поправкой на покупательную способность (PPP) и грейдами junior/middle/senior |

> Решения упражнений к проектам 16–21 находятся в `Projects/solutions/` (ноутбуки `16_Solutions.ipynb` … `21_Solutions.ipynb`).

## Требования к окружению

### Python библиотеки:
```bash
pip install numpy scipy pandas matplotlib seaborn jupyter
pip install torch torchvision  # Для Модуля 5
```

> Для проекта 18 (`18_IT_Job_Market_USA.ipynb`) живая загрузка данных идёт через бесплатный **U.S. Census ACS API** (`api.census.gov`). При отсутствии интернета используется встроенный снимок опубликованных значений.

### Запуск Jupyter Notebook:
```bash
jupyter notebook
```
